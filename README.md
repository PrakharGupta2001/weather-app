## About The Project

A simple and responsive Weather web app built using HTML, CSS and Javascript. It uses <a href="https://openweathermap.org/api">OpenWeatherMap API</a> to fetch Temperature, Weather, Humidity & Wind Speed details. It uses <a href="https://opencagedata.com/api">OpenCageData Geocoder API</a> to fetch exact location coordinates of the user. Built based on the 2021 UI trend 'Glassmorphism' . Hosted using <a href="https://www.netlify.com">Netlify</a>.

### Built With

* HTML & CSS
* Javascript
* Vanilla tilt.js library
* OpenWeatherMap & OpenCageData API
